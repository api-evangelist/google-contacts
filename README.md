# Google People API (Contacts)

The Google People API provides access to information about profiles and contacts. It allows you to list, create, update, delete, and search contacts, as well as manage contact groups. It replaces the legacy Google Contacts API and provides access to user profiles and directory information.

## Artifacts

- **OpenAPI**: [openapi/contacts.yml](openapi/contacts.yml) - OpenAPI 3.1.0 specification for the Google People API.
- **JSON Schema**: [json-schema/contacts.json](json-schema/contacts.json) - JSON Schema (draft 2020-12) for Google contact (person) resources.
- **JSON-LD**: [json-ld/contacts.jsonld](json-ld/contacts.jsonld) - JSON-LD context mapping Google People resources to Schema.org vocabulary.
